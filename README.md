## التشغيل

الموقع Static HTML - افتح `index.html` مباشرة في المتصفح أو ارفعه على أي استضافة.

### GitHub Pages

1. ارفع الملفات على GitHub
2. اذهب إلى Settings > Pages
3. اختر Branch: main
4. الموقع سيكون جاهز على `https://username.github.io/repo-name/`
